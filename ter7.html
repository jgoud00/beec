Circuit Fundamentals

üìç Node

A point in a circuit where two or more circuit elements meet. It's a junction point where current can divide or combine.

‚Ä¢ Connection point

‚Ä¢ Current distribution

‚Ä¢ KCL applies here

üåø Branch

A single element or group of elements connected between two nodes. Current flowing through a branch is the same at all points.

‚Ä¢ Between two nodes

‚Ä¢ Same current throughout

‚Ä¢ Can contain multiple elements

üîÑ Loop

‚Ä¢ Closed path

‚Ä¢ Returns to starting point

‚Ä¢ KVL applies here

Network Theorems

üîå Thevenin's Theorem

Any linear bilateral network can be replaced by a single voltage source (V_th) in series with a single resistance (R_th).

Steps to Find Thevenin Equivalent:

1. Remove the load resistor

2. Find V_th: Open circuit voltage across terminals

3. Find R_th: Replace all sources with their internal resistance

4. Draw equivalent circuit

5. Connect load and solve

Thevenin Equivalent Circuit:

I_L = V_th / (R_th + R_L)

üîÑ Norton's Theorem

Any linear bilateral network can be replaced by a single current source (I_N) in parallel with a single resistance (R_N).

Steps to Find Norton Equivalent:

1. Remove the load resistor

2. Find I_N: Short circuit current through terminals

3. Find R_N: Same as R_th

4. Draw equivalent circuit

5. Connect load and solve

Norton Equivalent Circuit:

I_L = I_N √ó R_N / (R_N + R_L)

Note: I_N = V_th / R_th and R_N = R_th

‚ö° Superposition Theorem

In a linear circuit with multiple sources, the response at any element is the sum of responses due to each source acting independently.

Application Steps:

1. Turn off all sources except one

2. Calculate response due to active source

3. Repeat for each source individually

4. Sum all individual responses

Note: Voltage source ‚Üí short circuit

Current source ‚Üí open circuit

Limitations:

‚Ä¢ Only applies to linear circuits

‚Ä¢ Cannot be used for power calculations directly

‚Ä¢ Dependent sources need special consideration

‚Ä¢ Each source must be independent
